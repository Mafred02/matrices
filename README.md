# Proyecto de matrices por Nicolás Rodríguez
Primero ingrese en cada rama y descargue el archivo .py que encuentre en cada una (entrada, main, menu y operaciones).
Una vez descargados, abra un nuevo notebook en Google Colab y cree una carpeta (puede ayudarse con la siguiente linea de codigo):
  import os
  os.makedirs("proyecto_matrices", exist_ok=True)
Una vez creada la carpeta dirijase al icono de carpeta que hay en el menu de opciones en el lado izquierdo de la pantalla y busque la carpeta. Una vez ahi, dele seleccione los tres puntos y luego en la opcion "subir", posteriormente suba cada archivo descargado. 
Por ultimo, importe el modulo main de la carpeta que creo y use la funcion "iniciar" para ver el programa en funcionamiento, (puede escribir algo como lo siguiente en Google Colab):
  from proyecto_matrices import main
  main.iniciar()
